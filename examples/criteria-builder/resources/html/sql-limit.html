Using CriteriaBuilder, you can limit the size of the result set in a very SQL-like way:

LIMIT {max}, {offset}

SELECT orderNumber
LIMIT 2, 2

Example

var cb = new CriteriaBuilder.Builder({
    store: mystore
});
// select 10 rows, starting at 5
var sql = 'select a.address addy, orderId, orderNumber num limit 10, 5';
// run
cb.query({sql:sql});
